HOW TO RUN THE PROJECT
1. 	Navigate to the akenjal1-p2 folder.
	
2. 	To compile and generate code-
		$ make

3.  To create server-
		$ ./server <port_number>

4.	Create mutiple servers and copy the ip-adress and port of each server to a text file on each line.

5.	Call init-
		$ ./init <text_file_containing_server_adresses>

6.	Running client-
		$ ./client1
		$ ./client
